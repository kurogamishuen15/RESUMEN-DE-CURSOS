MODULO 2 CLIENTES Y SERVIDORES

2.1

QUE SON LOS CLIENTES Y SERVIDORES


Los servidores son host con un software especial que les permite compartir almacenar o distribuir algún tipo de información a otros equipos conocidos como clientes , pueden alojar distintos servicios como paginas web o servidores de correos.

Actualmente las computadoras si les instalas el software correcto pueden actuar como servidor o como cliente. El cliente cuenta con un software especial que se conecta al servidor (Host principal) y les permite solicitar o mostrar la información del host principal. Un ejemplo de esto ese un navegador web.

ROLES DE CLIENTES Y SERVIDORES

Las pc conectadas a una red se conocen como host, los host pueden enviar y recibir información dentro de una red. Actualmente las computadoras pueden ser un servidor, un cliente o ambos.

Los servidores son hosts con software instalado que les permite proporcionar información, por ejemplo correo electrónico o páginas web, a otros hosts de la red. Cada servicio requiere un software de servidor independiente. Por ejemplo, para proporcionar servicios Web a la red, un host necesita un software de servidor Web. Cada destino que visita en línea es ofrecido por un servidor ubicado en alguna parte de una red que está conectada a Internet global.

REDES EN PARES

El software de servidor y el del cliente normalmente se ejecuta en pc separadas, pero también es posible que una sola pc actué como servidor y como cliente; a este tipo de red se le denomina P2P o peer to peer

La red mas simple consta de 2 computadoras conectadas con un cable de red o inalambrica, ambas actúan como clientes o servidor,ya que pueden recibir la información y también compartirla al mismo tiempo. Puedes crear un red P2P con mas dispositivos pero necesitaras un dispositivo de red intermedio.

Aplicaciones entre pares

Una aplicación P2P permite que un dispositivo funcione como cliente y como servidor dentro de la misma comunicación, como se muestra en la figura. En este modelo, cada cliente es un servidor y cada servidor es un cliente. Las aplicaciones P2P requieren que cada terminal proporcione una interfaz de usuario y ejecute un servicio en segundo plano.

Algunas aplicaciones P2P utilizan un sistema híbrido donde se descentraliza el intercambio de recursos, pero los índices que apuntan a las ubicaciones de los recursos están almacenados en un directorio centralizado. En un sistema híbrido, cada punto accede a un servidor de índice para obtener la ubicación de un recurso almacenado en otro punto.

MÚLTIPLES ROLES DE RED

Una computadora con software de servidor puede proporcionar servicios a múltiples clientes.

Una única pc puede contar con varios software de servido. En empresas pequeñas normalmente tienen un solo pc con maquinas virtuales que contienen varios software de servidor como email, archivos, webs etc.

Una única pc tambien puede contar con múltiples software de clientes, puede haber uno por cada servicio requerido y pueden conectarse a varios servidores al mismo tiempo, un ejemplo es abrir el navegador, posteriormente el drive y el correo al mismo tiempo.

2.2

SÍMBOLOS DE LA INFRAESTRUCTURA

INFRAESTRUCTURA DE RED

Es un conjunto de componentes físicos y lógicos interconectados que permiten intercambiar información entre si, están conformados por los siguientes dispositivos

Los dispositivos y los medios son los elementos físicos o hardware de la red. Por lo general, el hardware está compuesto por los componentes visibles de la plataforma de red, como una PC portátil, una PC, un switch, un router, un punto de acceso inalámbrico o el cableado que se utiliza para conectar estos dispositivos. A veces, puede que algunos componentes no sean visibles. En el caso de los medios inalámbricos, los mensajes se transmiten a través del aire mediante radio frecuencias invisibles u ondas infrarrojas.

Dispositivos finales (host): Son el origen o el destino de los datos transmitidos en la red. para identificar los host se utilizan las direcciones para saber que ruta de tomar el paquete

Dispositivos intermediarios: Son componentes físicos que logran conectar mas conectar mas dispositivos y gestionan la comunicación entre los mismos.

Medios de red : Es el canal o la carretera fisica por donde transita la información, puede ser visible como un cable o no visible como las ondas inalambricas.

2.3

SERVICIOS DE ISP

Las ISP o proveedores de servicio de Internet es el proveedor local de servios e infraestructuras en un sector, pueden ofrecer distintos servicios no solo de Internet como:

  • Canales de datos
  • Creación de redes locales
  • Servicios de Streaming
  • Telefonía fija
  • Hosting
  • Servicios de Email
  • Copia de respaldos
  • Enlaces punto a punto

Las ISP pueden tener infraestructura propia o alquilarla de otras empresas, son fundamentales ya que sus redes se conectan con otras isp para poder formar redes de redes (Internet), esta comunicación generalmente se da por cables troncales los cuales tiene una capacidad de transmisión grandisima para poder llevar la información, generalmente estos cables son de fibra optica y van por debajo del suelo o por mar.

CONEXIONES DE ISP

La interconexión de los ISP que conforman la red troncal de Internet es una red compleja de cables de fibra óptica con conmutadores y enrutadores de red costosos que dirigen el flujo de información entre los hosts de origen y de destino. Los usuarios domésticos promedio no conocen la infraestructura fuera de su red. Para un usuario doméstico, la conexión al ISP es un proceso bastante simple.

CONEXIONES DE CABLE COAXIAL Y DSL

Hoy en día aunque la fibra optica es económica, muchos de los proveedores utilizan la conexion por cable coaxial de cobre o DSL

Cable coaxial: Estos cables estaban diseñados para servicios de televisión sin embargo, al utilizar dos frecuencias distintas es posible transmitir televisión y internet de alta velocidad (Pero no simetrica) se utilizan modems de alta velocidad y divisores que separen la señal para la televiison

DSL: Es un cable con 3 terminales

1* Se utiliza para la telefonia. como estos servicios generalmente no separan las frecuencias se utiliza un hilo para transmitir la señal de telefono y evitar que se corte el internet.

2* Se utiliza un hilo solo para la descarga de la información o conocida como bajda.

3* Se utiliza para la subida de información (generalmente es mucho mas lenta que la bajda.

La conexión DSL aunque en la teoria cuenta con buena velocidad en la practica depende de la distancia, la calidad del cable etc etc

OTRO TIPO DE CONEXIONES:

Red de Celular:El acceso a Internet celular utiliza una red de telefonía celular para conectarse. En cualquier lugar donde tenga cobertura de telefonía móvil, puede tener acceso a Internet. El rendimiento se verá limitado por las capacidades del teléfono y la torre celular a la que se conecte. La disponibilidad del acceso a Internet por redes celulares es una gran ventaja para gente que vive en áreas sin acceso a otro tipo de conectividad a Internet, o para personas que van siempre de un lado a otro. La desventaja de la conectividad celular es que la empresa que presta el servicio suele medir el uso del ancho de banda de la conexión y puede cobrar tarifas adicionales por el ancho de banda que supere el plan de datos contratado.

Red Satelital: El servicio satelital es una buena opción para hogares u oficinas que no tienen acceso a DSL o cable. Las antenas parabólicas (ver la figura) requieren una línea de visión despejada hasta el satélite, por lo que podrían no ser adecuadas para zonas muy boscosas o lugares que posean algún otro tipo de obstrucción aérea. Las velocidades varían según el contrato, pero suelen ser buenas. Los costos de equipos e instalación pueden ser elevados (consulte con el proveedor para conocer las ofertas especiales) y luego se paga una tarifa mensual módica. Al igual que en el caso del acceso por redes celulares, la disponibilidad del acceso satelital a Internet es una gran ventaja para áreas que no tienen acceso a otro tipo de conectividad a Internet.

Telefono fijo: Una opción económica que utiliza cualquier línea telefónica y un módem. Para conectarse al ISP, el usuario llama al número telefónico de acceso del ISP. El bajo ancho de banda proporcionado por una conexión de módem de acceso telefónico generalmente no es suficiente para una gran transferencia de datos, aunque es útil para el acceso móvil mientras viaja. La opción de conexión de módem de acceso telefónico solo se debe considerar cuando no haya opciones de conexión más veloces disponibles.
